# Homebrew Security Tools Tap

Homebrew tap for security and compliance CLI tools.

## Installation

```bash
brew tap ethanolivertroy/sectools
```

## Available Tools

### cmvp

Terminal UI for searching NIST CMVP validated cryptographic modules.

```bash
brew install ethanolivertroy/sectools/cmvp
```

### kevs-tui

Terminal UI for browsing CISA Known Exploited Vulnerabilities (KEV) catalog with EPSS scores, CVSS data, and interactive charts.

```bash
brew install ethanolivertroy/sectools/kevs-tui
```

### fedramp

Terminal UI for browsing FedRAMP authorized cloud services.

```bash
brew install ethanolivertroy/sectools/fedramp
```

### vanta-exporter

CLI tool to export Vanta audit evidence organized by control.

```bash
brew install ethanolivertroy/sectools/vanta-exporter
```
